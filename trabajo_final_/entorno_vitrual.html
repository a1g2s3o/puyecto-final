Entorno vitrual

 

El entorno virtual es todo las herramientas que use la tecnología junto con el internet es puede ser desde bolg, YouTube, correo, redes sociales, etc. el punto que principal forma que distingue esta herramienta es la gran capacidad que tiene para obtener información de distiendan fuentes y de como representar esta información de una forma digerible para su entendimiento y aprendizaje.

Ejemplo de esto tenemos como un las cartulinas puedes convertirse en presentación de Power Point o otros programas, como temas complejos que se tenían que dibujar gráficos para entender se pueden reemplazar por animaciones para ver lo que esta ocurriendo.
